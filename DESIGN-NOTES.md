# Design notes

This version keeps the editorial, less-template-like structure while restoring a small amount of color and clearer interactive controls.

- Black and near-black surfaces remain the foundation.
- Sapphire blue and muted teal are the main accents, with restrained gold and plum used only where helpful.
- Project and photography filters use compact tab controls again.
- Social links and navigation states use subtle bordered buttons rather than plain text.
- Project categories and technology tags use low-contrast badges for easier scanning.
- There are no ambient glows, decorative grids, floating shapes, glass panels, or exaggerated animations.
- The layout remains responsive, including horizontally scrollable filters on small screens.


## Refined visible controls
- Home actions use distinct but restrained sapphire, teal, and gold treatments.
- Social links use subtle brand-aware borders rather than generic text links.
- Section labels and portrait metadata use compact tab styling for clearer hierarchy.


## Portrait caption
The homepage portrait uses a simple unboxed caption beneath the image, with the name on the left and location on the right.

## Project and award media
Project and award cards support optional multi-image carousels. They auto-advance slowly, pause during interaction, support keyboard controls and phone swipe gestures, and disable automatic motion when the visitor prefers reduced motion.


## Clean section titles
Decorative eyebrow pills above page and section headings were removed. Meaningful subsection labels now use simple text headings for a cleaner hierarchy.


## Latest refinement
- Removed the secondary description column from top-level page introductions.
- Moved the homepage portrait name and location beside the image on desktop and beneath it on mobile.
