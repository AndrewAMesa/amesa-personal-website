HOW TO ADD PHOTOS

1. Put image files in this folder, for example:
   public/photography/idaho-sunset.jpg

2. Open data/site.ts.

3. Find the photographs array and change a photo's src from an empty string to:
   /photography/idaho-sunset.jpg

4. Update the title, location, alt text, category, and orientation.

Recommended: optimized JPG, WebP, or AVIF files around 1800-2400 px on the long side.
