# Content summary

This version retains the complete professional content set and adds responsive image-carousel support for projects and awards.

- Projects: optional one-image display or multi-image carousel
- Awards: optional one-image display or multi-image carousel
- Controls: arrows, position dots, keyboard navigation, and phone swipe
- Motion: slow automatic rotation with pause-on-interaction and reduced-motion support
- Headshot: homepage portrait with clean name and location caption
- Photography: separate filtered portfolio page
- Contact: LinkedIn, GitHub, YouTube, Instagram, email, and optional Calendly
